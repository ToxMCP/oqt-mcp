# Code of Conduct

## Our commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and professional in discussions and reviews.
- Assume good intent and provide constructive feedback.
- Focus on technical issues, not individuals.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting language, or deliberate disruption.
- Sharing private information without explicit permission.

## Scope

This Code of Conduct applies to project spaces (issues, pull requests, discussions, and related community channels).

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

Report unacceptable behavior to `oss@senseibelbi.dev`.
